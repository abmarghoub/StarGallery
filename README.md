## 1. Objectif du TP

- Comprendre et appliquer les concepts de RecyclerView pour afficher une liste dynamique d’éléments.
- Apprendre à manipuler des objets Java et à gérer des données via un service centralisé.
- Mettre en place des popups interactifs pour noter et supprimer des éléments.
- Implémenter une recherche en temps réel avec SearchView pour filtrer une liste.
- Personnaliser l’interface utilisateur avec des couleurs, icônes et étoiles pour améliorer l’expérience utilisateur.
- Développer une application Android complète combinant affichage, interaction utilisateur et gestion des données.

---

## 2. Architecture du TP

### 2.1 Stack technologique

| Technologie                         | Rôle dans le projet                          |
| ----------------------------------- | -------------------------------------------- |
| **Java**                            | Langage de programmation principal           |
| **Android Studio**                  | IDE pour le développement Android            |
| **RecyclerView**                    | Affichage dynamique de la liste de joueurs   |
| **AlertDialog**                     | Popups pour noter et supprimer un joueur     |
| **Glide**                           | Chargement des images depuis Internet        |
| **ConstraintLayout & LinearLayout** | Mise en page flexible et responsive          |
| **CircleImageView**                 | Affichage des images des joueurs circulaires |
| **SearchView**                      | Recherche instantanée dans la liste          |
